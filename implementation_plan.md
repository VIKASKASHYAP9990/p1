# Sales Data Analysis Dashboard

## Overview
An interactive, browser-based Sales Performance Dashboard built with HTML, CSS (Vanilla), and JavaScript.
Uses Chart.js for rich visualizations and embedded Superstore-style data.

## Components
- index.html — main dashboard shell
- data.js — embedded sales dataset (500+ records, Superstore-style)
- dashboard.js — data processing, filtering, chart rendering logic
- style.css — dark glassmorphism premium design system

## Features
- KPI Cards: Total Sales, Total Profit, Profit Margin, Order Count
- Bar Chart: Top 10 Products by Sales
- Line Chart: Monthly Sales & Profit Trends
- Pie/Donut Chart: Category Distribution
- Horizontal Bar: Regional Performance
- Interactive Filters: Region, Category, Year
- Animated transitions and hover effects
